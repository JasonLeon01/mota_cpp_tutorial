Mota_cpp
=========

**Mota_cpp** 是一个基于C++语言和SFML图形库的魔塔类游戏框架。

主要特点
--------

**画面**

支持640x480的窗口和至多其长宽 **2倍** 大小的分辨率

**音乐**

使用的是SFML的Audio库，目前支持的音乐播放类型为wav, ogg/vorbis和flac。

**编程语言**

基于C++语言和SFML图形库，当前采用的是包含C++20在内的特性，您可以通过IDE调整至最新标准。
